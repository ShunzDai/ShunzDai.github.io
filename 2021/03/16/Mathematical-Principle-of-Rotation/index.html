空间旋转的数学原理

Date: 三月 16, 2021 22:27:56  Author: Shunz Dai  Category: To be continue...  本文阅读量:次

笔者这些日子一直在学四元数Kalman滤波，看的文献有工程师写的，也有数学、物理工作者写的.读书的时候笔者就曾听说过四元数，在现代物理里，四元数似乎并没有起到很大的作用，物理工作者更多谈及的是\(SU(2)\)的Lie代数\(\mathfrak{su}(2)\).但笔者发现，若将\(\mathfrak{su}(2)\)表述为四元数的形式，似乎可以得到一些有意思的结论.这里笔者就讲一讲读文献的时候想到的、文献里大抵没有的东西.

约定

首先约定一些符号表征方法.笔者使用西文字母\(a,b,c,\cdots\)作为取值为\(1,2,3\)的抽象指标，使用希腊字母\(\mu,\nu,\rho,\cdots\)作为取值为\(0,1,2,3\)的抽象指标，抽象指标之间使用Einstein求和约定，即相同指标表示求和.严格的讲，使用Einstein求和约定时必须是一对上下指标表示求和.但本文将放宽这一约定，所有指标均使用下标，不再考虑上下指标的差异.在不加声明的情况下读者应能自行分辨求和对象和求和范围，笔者不再额外声明.

然后是一些常用记号.

  • \(I\)：单位矩阵.在不加声明的情况下读者应能自行分辨其维数，笔者不再额外声明.

  • \(\delta_{\mu\nu}\)：Kronecker符号，其定义如下： \[\begin{equation}\begin{split} \delta_{\mu\nu} = \begin{cases} 1 & When\ \mu = \nu;\\ 0 & When\ \mu \neq \nu. \end{cases} \end{split}\end{equation}\]

  • \(\varepsilon_{abc}\)：Levi-Civita符号，其定义如下： \[\begin{equation}\begin{split} \varepsilon_{abc} = \begin{cases} -1 & When\ \mu\nu\rho\sigma\ are\ odd\ permutation;\\ 0 & When\ \mu\nu\rho\sigma\ are\ not\ a\ permutation;\\ 1 & When\ \mu\nu\rho\sigma\ are\ even\ permutation. \end{cases} \end{split}\end{equation}\] \(\varepsilon_{\mu\nu\rho\sigma}\)的定义与之相同.

  • \(\sigma_a\)：Pauli矩阵，其定义如下： \[\begin{equation} \sigma_1 = \left(\begin{array}{cccc} 0&1\\ 1&0 \end{array}\right); \sigma_2 = \left(\begin{array}{cccc} 0&-i\\ i&0 \end{array}\right); \sigma_3 = \left(\begin{array}{cccc} 1&0\\ 0&-1 \end{array}\right). \end{equation}\] 容易证明Pauli矩阵具有如下对易关系(多层内积空间与泡利矩阵(Pauli matrices)的运算性质)： \[\begin{equation}\begin{split} \left[\sigma_a,\sigma_b\right] = \sigma_a\sigma_b - \sigma_b\sigma_a = 2i\varepsilon_{abc}\sigma_c; \end{split}\end{equation}\] \[\begin{equation}\begin{split} \left\{\sigma_a,\sigma_b\right\} = \sigma_a\sigma_b + \sigma_b\sigma_a = 2\delta_{ab}I. \end{split}\end{equation}\] 从以上两则关系可以推出 \[\begin{equation}\begin{split} \sigma_a\sigma_a = I; \end{split}\end{equation}\] \[\begin{equation}\begin{split} \sigma_a\sigma_b = \delta_{ab}I + i\varepsilon_{abc}\sigma_c. \end{split}\end{equation}\]

\(SO(3)\)群

\(SO(3)\)群的定义

\(SO(3)\)群是一般线性群\(GL(3,\mathbb{R})\)的一个子群（这意味着\(SO(3)\)的元素是可逆的），它描述了3维空间的转动，其定义如下： \[\begin{equation}\begin{split} SO(3) = \left\{ R\in GL(3,\mathbb{R}): R^T = R^{-1},det(R) = 1\right\}, \end{split}\end{equation}\] 其中\(R^T = R^{-1}\)将\(GL(3,\mathbb{R})\)的群元限制在正交子群\(O(3)\)内，使矩阵只产生旋转，不会对空间进行拉伸或者压缩，而\(det(R) = 1\)使得矩阵保定向，不会将右手系变成左手系.对于任意的\(R\in SO(3)\)，虽然其拥有9个矩阵元，但上述定义给出了正交性要求\(R^T = R^{-1}\)，对任意\(R\in SO(3)\)，正交性使得9个矩阵元满足9个方程\(R_{\mu\nu} R_{\rho\nu} = \delta_{\mu\rho}\)，由于Kronecker符号的对称性，这9个方程中相互独立的方程只有3(3-1)/2=6个，9个矩阵元要满足6个约束条件，因此独立矩阵元的数目实际只有3个.这三个矩阵元分别对应了绕三个正交坐标轴的旋转变换，我们用Euler角描述这一性质.

Euler角

规定3个Euler角分别是横滚角(Roll)，俯仰角(Pitch)，航向角(Yaw)，它们分别对应绕\(X,Y,Z\)轴的旋转.

这样规定是因为在工程项目中，载具通常使用前左上(FLU)坐标系，即载具机头朝前时，载具前、左、上三个方向分别为\(X,Y,Z\)轴正方向.在这种坐标系中，横滚角描述了载具立轴与铅垂线的夹角，这是一个绕\(X\)轴的转动；俯仰角描述了载具纵轴与水平面的夹角，这是一个绕\(Y\)轴的转动；航向角描述了载具纵轴与子午面的夹角，这是一个绕\(Z\)轴的转动.

规定绕\(X,Y,Z\)轴逆时针旋转基底（坐标系）的变换矩阵分别为\(R_X(r),R_Y(p),R_Z(y)\)，顺时针旋转基底（坐标系）的变换矩阵为前3个矩阵的逆变换（这些矩阵需要从左向右作用在基底列上）.通过简单的计算可以得到： \[\begin{equation}\begin{split} R_X(r)&= \left(\begin{array}{ccc} 1 &0 &0 \\ 0 &\cos r &\sin r \\ 0 &-\sin r &\cos r \end{array}\right), R_X^{-1}(r)=R_X^{T}(r)= \left(\begin{array}{ccc} 1 &0 &0 \\ 0 &\cos r &-\sin r\\ 0 &\sin r &\cos r \end{array}\right);\\ R_Y(p)&= \left(\begin{array}{ccc} \cos p &0 &-\sin p \\ 0 &1 &0 \\ \sin p &0 &\cos p \end{array}\right), R_Y^{-1}(p)=R_Y^{T}(p)= \left(\begin{array}{ccc} \cos p &0 &\sin p \\ 0 &1 &0 \\ -\sin p &0 &\cos p \end{array}\right);\\ R_Z(y)&= \left(\begin{array}{ccc} \cos y &\sin y &0 \\ -\sin y &\cos y &0 \\ 0 &0 &1 \end{array}\right), R_Z^{-1}(y)=R_Z^{T}(y)= \left(\begin{array}{ccc} \cos y &-\sin y &0 \\ \sin y &\cos y &0 \\ 0 &0 &1 \end{array}\right). \end{split}\end{equation}\] 通过上述3个基础旋转可以组合得到3维空间中的任意旋转，要注意的是旋转顺序不可交换，有效的旋转顺序一共有12种，本文采用\(ZYX\)旋转顺序.\(ZYX\)顺序的旋转矩阵是上述3个逆时针旋转\(R_X\)，\(R_Y\)，\(R_Z\)的复合，它们按从左到右的顺序依次作用到坐标系的基底列上 \[\begin{equation}\begin{split} R(r,p,y):&=R_XR_YR_Z= \left(\begin{array}{ccc} \cos p\cos y &\cos p\sin y &-\sin p \\ \sin r\sin p\cos y-\cos r\sin y &\sin r\sin p\sin y+\cos r\cos y &\sin r\cos p\\ \cos r\sin p\cos y+\sin r\sin y &\cos r\sin p\sin y-\sin r\cos y &\cos r\cos p \end{array}\right). \end{split}\end{equation}\] 当旋转对象是参考向量的坐标时，则对于同样的旋转矩阵，坐标的旋转方向刚好与基底的旋转方向相反.

生成元

生成元的定义是Lie群在单位元处的切空间.根据单位元的定义，容易判断\(SO(3)\)的单位元是单位阵\(I\)，此时Euler角参数均为0.对\(R(r,p,y)\)求偏导数就得到了生成元： \[\begin{equation}\begin{split} \varepsilon_1 &= \frac{\partial R(r,p,y)}{\partial r}\Bigg|_{(0,0,0)} = \left(\begin{array}{ccc} 0 &0 &0 \\ 0 &0 &1 \\ 0 &-1 &0 \end{array}\right);\\ \varepsilon_2 &= \frac{\partial R(r,p,y)}{\partial p}\Bigg|_{(0,0,0)} = \left(\begin{array}{ccc} 0 &0 &-1\\ 0 &0 &0 \\ 1 &0 &0 \end{array}\right);\\ \varepsilon_3 &= \frac{\partial R(r,p,y)}{\partial y}\Bigg|_{(0,0,0)} = \left(\begin{array}{ccc} 0 &1 &0 \\ -1 &0 &0 \\ 0 &0 &0 \end{array}\right). \end{split}\end{equation}\] 用上述3个生成元可以张成一个线性空间\(\mathfrak{so}(3)\)，它是Lie群\(SO(3)\)的Lie代数的一个表示，称为基本表示.容易证明\(\varepsilon_{a}\)满足结构方程 \[\begin{equation}\begin{split} \left[\varepsilon_a,\varepsilon_b\right] = \varepsilon_{abc}\varepsilon_c, \end{split}\end{equation}\] 其中\(\varepsilon_{abc}\)是\(SO(3)\)的结构常数.通过伴随作用\(ad:\mathfrak{so}(3)\rightarrow End(\mathfrak{so}(3))\)可以得到伴随表示 \[\begin{equation}\begin{split} ad_{\varepsilon_a}\varepsilon_b = \varepsilon_{abc}\varepsilon_c \Rightarrow ad_{\varepsilon_a} = \varepsilon_{abc} = \varepsilon_a\in End(\mathfrak{so}(3)), \end{split}\end{equation}\] 这表示\(\mathfrak{so}(3)\)的基本表示与伴随表示等同.

四元数乘法群

四元数的定义

记\(g_0 =I,g_a =-i\sigma_a\)表示四元数的基底（这里只是借用线性代数的名词，虽然以\(g_\mu\)为基底确实构成了一个线性空间，但我们并不会使用这一性质），则\(\mathbb{H}=\{A=A_\mu g_\mu:A_\mu\in\mathbb{R},\mu=0,1,2,3\}\)，表示全体四元数构成的集合.

\(\mathbb{H}\)上的群结构

\(\mathbb{H}\)上具有一种乘法，具备以下特性：

  1. 封闭性，根据Pauli矩阵的对易关系 \[\sigma_a\sigma_b=i\varepsilon_{abc}\sigma_c+\delta_{ab}I\] 可以得到 \[g_a g_b=\varepsilon_{abc}g_c-\delta_{ab}g_0,\] 于是对于任意\(A,B\in \mathbb{H}\)都有 \[\begin{equation}\begin{split} AB&=(A_0 g_0 + A_a g_a)(B_0 g_0 + B_b g_b)\\ &=A_0 B_0 g_0 g_0 + A_a B_0 g_a g_0 + A_0 B_a g_0 g_a + A_a B_b g_a g_b\\ &=(A_0 B_0 - A_aB_a)g_0 + (A_0 B_c + A_c B_0 + A_a B_b \varepsilon_{abc})g_c, \end{split}\end{equation}\] 若记\(C_0=A_0 B_0 - A_aB_a\)，\(C_c= A_0 B_c + A_c B_0 + A_a B_b \varepsilon_{abc}\)，则上式可写成\(AB=C\)，其中\(C=C_\mu g_\mu\)显然也是\(\mathbb{H}\)的元素，这就证明了封闭性.

  2. 定义\(\mathbb{H}\)上的共轭映射 \[\begin{equation} *:\mathbb{H}\rightarrow \mathbb{H};A=A_\mu g_\mu\mapsto A^*=A_0g_0-A_ag_a, \end{equation}\] 则\(*\)诱导了\(\mathbb{H}\)上的\(L_2\)范数 \[\begin{equation} (\cdot,\cdot):\mathbb{H}\rightarrow \mathbb{R};(A,A)=||AA^*||^\frac{1}{2}. \end{equation}\] 用1中的运算可以验证\((\cdot,\cdot)\)确实是范数： \[\begin{equation}\begin{split} AA^*&=(A_0 A_0 + A_aA_a)g_0 + (-A_0 A_c + A_c A_0 + A_a A_b \varepsilon_{abc})g_c\\ &=A_\mu A_\mu g_0, \end{split}\end{equation}\] 于是\(||AA^*||^\frac{1}{2}=(A_\mu A_\mu)^\frac{1}{2}\)，这显然满足范数的非负性、齐次性和三角不等式.

  3. 结合律，\(A(BC) = (AB)C;\)

  4. 单位元，即\(A=g_0\)；

  5. 逆元，由范数诱导\(A^{-1}=\frac{A^*}{||AA^*||}\).

现在将注意力放到\((5)\)上，我们引入一个\(4\times 4\)的反对称矩阵\(F(A)\)，其第\(\mu\)行第\(\nu\)列的元素为\(F_{\mu\nu}(A)\).命： \[\begin{equation}\begin{split} F_{a0}(A) = -F_{0a}(A) = A_a;F_{ab}(A) = \varepsilon_{abc}A_c;{}^*F_{\mu\nu}(A)=\frac{1}{2}\varepsilon_{\mu\nu\rho\sigma}F_{\rho\sigma}(A). \end{split}\end{equation}\] 使用这套记号，\((xx)\)可以改写为： \[\begin{equation}\begin{split} AB&=(A_0 B_0 - A_aB_a)g_0 + (A_0 B_c + A_c B_0 + A_a B_b \varepsilon_{abc})g_c\\ &=(A_0 B_0 g_0 + A_0 B_c g_c) + (- A_a B_a g_0) + (A_c B_0 g_c) + (A_a B_b \varepsilon_{abc} g_c)\\ &=A_0 B + F_{0a}(A) B_a g_0 + F_{a0}(A) B_0 g_a + F_{ab}(A) B_{b} g_a\\ &=A_0 B + F_{0a}(A) B_a g_0 + F_{a\nu}(A) B_\nu g_a\\ &=A_0 B + F_{\mu\nu}(A) B_\nu g_\mu, \end{split}\end{equation}\] 其中用到了重复指标可以成对替换，而最后一个等号用到了\(F_{aa} B_a g_a = 0\).工程师朋友可能不熟悉这套语言，但对物理工作者而言，\(F_{\mu\nu}\)可再熟悉不过了，它对应了某个主丛的曲率，而\({}^*F_{\mu\nu}\)是\(F_{\mu\nu}\)的Hodge对偶.
